Tailwind CSS Demo Project

This page will show you how to style your Spring Boot application with Tailwind CSS

Creating your Spring Boot project

We have created the Spring Boot project with Maven for this demo.

Dependencies needed: Spring Web, Thymeleaf

Adding Tailwind CSS

Once the project is created, we can install Tailwind CSS via npm.

Installation process

  1. Create npm project in the root folder

  2. Install tailwindcss and @tailwindcss/cli via npm

  3. Create npm project (e.g. in the root folder)

  4. Create the app-style.css file in the resources/static/css folder

Build process

Add the @import "tailwindcss"; import to the app-style.css file.

Create the "build" script for the Tailwind CLI build process in the package.json file.

"build": "npx @tailwindcss/cli -i ./src/main/resources/static/css/app-style.css -o ./src/main/resources/static/css/frontend-style.min.css --minify

This will scan our source files for classes and will build our CSS, creating the frontend-style.min.css minified CSS file.

Additionally, we can create the "watch" script for development to rebuild the non-minified built CSS file based on any change in our source files.

"watch": "npx @tailwindcss/cli -i ./src/main/resources/static/css/app-style.css -o ./src/main/static/resources/css/frontend-style.min.css --watch"

The source files to scan can be set in our app-style.css file:

        @import "tailwindcss";
        @source "../java/**";

Important: don't forget to add the link to frontend-style.min.css (built CSS) to the HTML files.

In case we want to build the CSS file from maven instead of manually, we can use the frontend maven plugin
